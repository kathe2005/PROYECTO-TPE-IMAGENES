# PROYECTO TPE - GESTIÓN DE IMÁGENES 
Este es el sistema central para el manejo de evidencias visuales 

## Tecnologías 
* **TypeScript:** El lenguaje base. 
* **Node.js & Express:** El motor.  
* **MongoBD:** El archivo confidencial de datos. 
* **Sharp:** Para el procesamiento táctico de imágenes. 

## Instalación 
1. Clonar el repositorio. 
2. Ejecutar `npm install` para reclutar las dependencias. 
3. Configurar el archivo `.env` con tus credenciales. 
4. Iniciar el servidor con `npm run dev` (o el comando que definas)
